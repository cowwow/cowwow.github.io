收获蚁

🔥收获蚁饲养指南🔥
🔥收获蚁饲养指南🔥原生🔥大头🔥红胸🔥肯尼亚🔥工匠🔥

✅.适宜温度
爆产温度29-30°C，原生收获蚁最高温33℃，肯尼亚收获蚁最高温30℃，高温预警标志是收获蚁搬蛹幼去喂水处乘凉，这是大部分蚂蚁的高温预警动作。低温预警温度大概为15℃。

✅.湿度偏好
收获蚁喜好偏干，除了新后堵水裸试管外，群落一般直接裸试管养即可，铺地试管不需要加水。

✅.蚁巢布置要求
1⃣️推荐不铺地试管巢，新后堵水试管，群落干裸试管；而且收获蚁发育快，试管拓展比较方便。
2⃣️活动区尽可能大面积喂水，收获蚁湿度要求不高但屁股不能储水，所以要经常喝水；
3⃣️活动区尽可能大，而且要有铺地，收获蚁容易在活动区角落发粪涂墙，大活动区容易清理垃圾。

✅.开挂合蛹
1⃣️切记！新后孤后合蛹前确定已喂蛋白质已喂饱，确定是在养定的环境中。要不然整什么都没用。同种蛹照样失败，甚至老娘暴毙
2⃣️记得是合蛹不是合幼，幼虫合进去需要喂会消耗蚁后营养，合幼也不是不行，如果合幼进去就要勤喂食。纯蛹优势明显！
3⃣️合蛹前晾吹风一小时左右散气味，主要是散掉运输中的塑胶味，泡沫味，然后直接丢进活动区或新后试管口
4⃣️整个合蛹过程在1小时到1天不等，蛹来来回回搬到垃圾区都正常，让蛹飞一会，放个布遮个光睡一觉，它们会逐渐接受的。合蛹成功标志为蛹搬到了蚁后脚下。如果一天以后还在垃圾区，那就是不接受了。
5⃣️不要喂种子，小群没大工剥皮，也没蛋白质！

✅.以上适合肯尼亚50工以内的小群落，或者小型收获蚁100工以内的群落，随着群落的变大，合蛹成功率会越来越低。当群落大于50时，方法要改。能提升蛹的接受率，降低损耗的个人经验如下。
1⃣️先层别即将出工的脚在动的黑蛹或者已经出工的黄脑袋工，先合。可以有效避免群落吃合的蛹以及蛹出工几天就死亡的问题(工蚁嘴欠，喜欢无聊来一口)。
2⃣️余下的蛹，常温存放，等蛹发黑或 者出工，再重复上述方法，直到全部合完
3⃣️如果黑蛹现场被开膛破肚，黄脑袋工当场立刻被秒杀，基本断定合不进去了

  1. 适宜温度29°C
    野蛮最高温33°C，肯尼亚最高温30°C，高温预警标志是收获蚁搬蛹幼去喂水处乘凉,这是大部分蚂蚁的高温预警动作。低温预警温度大概为15°C。
  2. 湿度表现为偏干
    除了新后堵水裸试管外，群落-般直 接裸试管养即可，想要铺地试管也可但不能加水,强烈不建议具支铺地试管加水。
  3. 蚁巢布置要求
    ①推荐不铺地试管巢，新后堵水试管，群落干裸试管;而且收获蚁发育快，试管拓展比较方便。
    ②活动区尽可能大面积喂水，收获蚁湿度要求不高但屁股不能储水，所以要经常喝水;
    ③活动区尽可能大，而且要有铺地，收获蚁容易在活动区角落发粪涂墙，大活动区容易清理垃圾。
  4. 新后开挂合蛹
    ①新后合蛹前蚁后先喂饱
    ②记得是合蛹不是合幼，幼虫合进去需要喂会消耗蚁后营养，合幼也不是不行，如果合幼进去就要勤喂食。
    ③合蛹前晾吹风一小时左右散气味，然后直接丢进去，蚁后会惊慌失措(异常兴奋)然后赶快搬蛹。
    ④合蛹一-般比较科学的是先合10个左右，待 出工后合50 ~ 100蛹
  5. 注意事项
    ①新后不要给种子，- -般不会吃的，破10工再给
    ②大群落活动区要经常定期补防逃粉

大头蚁

  1. 大头蚁需要湿度比较高，会把卵幼放在高湿区，工蚁站位在干区，意味着巢室内部最理想就是干湿分离，而且密集干湿分离。
  2. 大头蚁容易爆产，但体型小，所以巢室一般都是多个小扁室。
  3. 大头蚁活动区要足够大，不然清理垃圾麻烦。
  4. 大头蚁一般保持适度的巢大蚁少会好一些，如果巢小蚁多会死工严重。
  5. 大头蚁不能频繁打扰，会减少产卵量,广大头尤其明显。

弓背蚁

🔥黑金弓背蚁饲养指南🔥
✅适宜温度
黑金弓背蚁适宜温度28°C-32°C，高温预警标志是弓背蚁搬茧子、幼去喂水处乘凉。

✅湿度喜好
1⃣️新后可以用18口径堵水试管饲养。堵水建议用PVA堵水棉，更加干净；同时因为前期发育缓慢，可以堵水三分之一
2⃣️群落整体喜欢偏干，巢体过湿，会抑制蚁后产卵甚至起霉，莫名其妙死工（多个黑金大佬已验证）

✅蚁巢推介
1⃣️推荐试管巢，试管口径18mm或20mm，黑金喜欢偏拥挤的环境，蚁后更有安全感。新后堵水试管，群落百工以上干裸试管或者简单铺地试管（黑色、红色铺地视觉效果更好），
2⃣️混凝土/石膏/亚克力平面巢都可以养黑金，但是不方便偷茧子，需要调整巢体与活动区温湿度差，引导工蚁主动把茧子搬到活动区晾茧子。

✅糖的危害
黑金养的好不好，除了强后、弱后之分外，最关键的就是喂食，直接影响群落发育速度。其喂食难度主要是小群（0-300工），吃的少，需要严格控制糖分摄入。不要喂糖水、营养液，理由如下
1⃣️弓背蚁极度嗜糖，糖水喝饱后，进食蛋白质的量会大大减少。
2⃣️长期喂食糖水，会因为糖水中蛋白质含量少，导致群落发育慢，甚至蚁后绝育。
3⃣️糖液喝多了，工蚁容易在巢室内吐水。

✅日常喂食
1⃣️新后阶段
水：新后用堵水试管。百工后可以纯干养，活动区放喂水器
蛋白质：新后到家/出幼/出工各喂一次（切断樱桃、面包虫），喂完1天后拿出
日常糖源：冰糖挂机
2⃣️50-300工
相对随意，蛋白质为主，冰糖为辅，水常供，仍然禁用营养液
3⃣️300工以上
你已经是个老油条了，还看个毛线攻略

✅目前已知的几个群落级别
LV1 新后 慢
LV2 100工 普通
LV3 300工 高产
LV4 1000工 爆产
LV5 3000工 爆产
LV6 3000工以上 不知道没见过

✅新后开挂合蛹
1⃣️新后合蛹前蚁后先喂饱
2⃣️合蛹前晾吹风一小时左右散气味，然后直接丢进去，蚁后会惊慌失措（异常兴奋）然后赶快搬蛹。
3⃣️合蛹一般比较科学的是分批合，比如买了100个，就50个50个的倒，让蚁后适应
4⃣️黑金弓背蚁任何群落大小可以合茧子，而且只要蚁后信息素压的住，能明显的提升蚁后的产卵速度

  • 费氏弓背蚁

第一，新后饲养操作

  1. 新后用15口径堵水试管养，温度28 ~32 (有条件就30)，到家后喂一 次半截1cm的樱桃蟑螂或杜比亚蟑螂，48小时后清干净。
  2. 出幼虫的时候喂半截蟑螂，48小时清干净。
  3. 出工后每隔三天半只蟑螂。
  4. 破10工后每天半只蟑螂，开活动区，活动区放干冰糖和喂水。
  5. 规模50工~ 100工每天- -只剪断小蟑螂

第二，巢环境要求

  1. 有铺地的更好，铺地不要弄湿。
  2. 听说巢内最好有透气缝，透气更利于发育。
  3. 尽可能高温(31度)， 但 高温水分蒸发严重，注意补水。
  4. 最理想就是高温31度，空气干燥(冰糖不会融化)，水源充足，有遮黑。
    第三，饲养禁忌
  5. 切记不要喂糖水，营养液等会改变蚂蚁口味，还会粘死工蚁。想玩色素的就当我没
  6. 注意一下活动区有没有出现白色细小的啮虫，这玩意会让黑金减育。
    第四，开挂操作
  7. 只推荐合茧子，不推荐锁后合巢!不解.释，实践出真知。
  8. 红金更容易接受茧子。
  • 其他弓背蚁
  1. 拟光腹弓背蚁和尼科巴弓背蚁比较容易
  2. 拟光腹28°C，尼科巴28°C，卡琳弓背/拟大头弓背26°C。。。。

猛蚁

  1. 介质保湿干湿分离的巢环境
  2. 除了猎镰聚纹短猛需要防逃，其余大部分都不用。
  3. 结茧介质需要红土，推荐红土粉或者椰土，建议直接铺在巢内。
  4. 控制湿度标准如下:介质深色不渗水，红土深色不渗水，红土松软不结块。
  5. 满3个月换- -次巢，无他，长期保湿会产生异味会出异常。
  6. 适宜温度:一般在26°C~ 28°C

举腹蚁

🔥棕头举腹蚁饲养指南🔥
✅适宜温度
爆产温度28-32℃，26-27℃蛹幼发育开始减慢，新后温度28-30℃
✅新后注意事项
1⃣️到货大概为1-3月，接下来的4-6月会一直有售
2⃣️新后1周喂1次蛋白质，尽量昆虫蛋白质，第2天清理出来
3⃣️喂糖水，注意糖水粘度，或者使用棉球蘸取糖水，要不然工蚁成片的粘死（大群一样死一片）
4⃣️50后左右可以开活动区了
✅湿度要求
喜欢干燥，新后用15口径试管堵水1/5（蚁后比较小）
✅巢穴推介
只推介试管巢
1⃣️棕头喜欢普通的试管，不喜欢铺地，喜欢把卵幼蛹都粘到光滑的表面上
2⃣️棕头体型偏小，群落成规模后建议使用18口径、20口径饲养
3⃣️棕头举腹蚁不怕光不怕震抓地强吸不动，卵幼粘性很大，如果使用石膏巢饲养无论是换巢、偷蛹都是地狱级，
✅日常注意事项
1⃣️食物：棕头可以当食肉蚁养，是肉来者不拒，群落养好的标志是囤积肉粒，代表他们吃饱了，也是爆产起飞的标志，否则请查明原因
2⃣️水源：棕头牙口极好，软的石膏、胶塞、棉棒、pva这些全都可以非常快速的咬穿，所以特别注意堵水试管，否则水淹七军（棕头蠢基本必死），建议使用带网格的喂水器，或者巢室中加入一根巨支铺地试管用于取水。其他都不建议，因为他们见啥咬啥
3⃣️防逃：棕头趴壁抓地力很强，但欲望不强，防逃粉可以用很久。
4⃣️冰糖：棕头嗜糖，前期要严格控制，否则不会出现肉末，不产卵会绝育，大群随意
5⃣️其他：巢室与活动区连接管、各种开孔处胶塞一定要定期检查，咬穿不开玩笑
6⃣️其他：从1000工以后，棕头发育成几何倍增长，基本上老娘吃一口下一个蛋
✅棕头合蛹
1⃣️没有太多讲究，刚出生的黄头工、常规蛹都可以合，接受度几乎为100%
2⃣️蛹到家先分类，优先合掉黄蛹，等黄蛹出工后再把其余白蛹合进去，可以减轻蚁后压力
3⃣️老生常谈，合蛹之前请把蚁后喂饱，以免节外生枝
